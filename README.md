# Obsidian 自定义排序文件夹插件

> Obsidian 知识库自定义排序视图插件

---

## 功能特性

### 核心功能

1. **可视化拖拽排序** - 在树形视图中直接拖拽调整文件/文件夹顺序
2. **自定义图标** - 支持 emoji 和自定义图片作为文件夹/文件图标
3. **多级目录支持** - 支持任意深度的子文件夹排序
4. **自动保存** - 拖拽后自动更新 `sortspec.md` 文件
5. **实时监听** - 文件变化自动刷新视图
6. **文件夹笔记** - 点击有同名 `.md` 的文件夹自动打开该文件

### 拖拽排序规则

| 操作 | 效果 |
|------|------|
| 同文件夹内拖拽 | 调整排序顺序 |
| 拖入展开的文件夹上方/下方 | 移动到该文件夹 |
| 跨文件夹拖拽 | 移动到目标位置 |

### 图标功能

- **系统 emoji** - 内置 16 个分类的常用 emoji
- **自定义图片** - 支持上传 SVG/PNG/JPG 等图片作为图标
- **最近使用** - 自动记录最近使用的图标

## 快速开始

### 安装插件

1. 在 Obsidian 中打开 **设置** → **社区插件**
2. 搜索 "Custom Sort Folder View"
3. 安装并启用插件

### 打开视图

1. 点击左侧 Ribbon 区域的插件图标
2. 或使用命令面板搜索 "自定义排序"

## 使用示例

### 基本拖拽排序

```
原始顺序:                    拖拽后:
├─ 文件夹 A                 ├─ 文件夹 A
├─ 文件夹 B                 ├─ 文件夹 C
├─ 文件夹 C                 └─ 文件夹 B
└─ 文件 A
```

### 移动文件到文件夹

```
原始状态 (文件夹已展开):
├─ 📁 项目
│   └─ 子笔记.md
└─ 📄 笔记.md

拖拽 "笔记" 到文件夹内:
└─ 📁 项目
   ├─ 📄 笔记.md  ← 新移入
   └─ 子笔记.md
```

### 自定义图标

1. 右键点击文件夹或文件
2. 选择 "设置图标"
3. 选择 emoji 或上传自定义图片
4. 目录树中显示自定义图标

## 排序配置

### sortspec.md 格式

每个文件夹的 `sortspec.md` 文件控制该文件夹的排序：

```yaml
---
sorting-spec:
  - 文件夹 A
  - 文件夹 B
  - 文件 1
  - 文件 2
custom-icons:
  文件夹 A: 🗂️
  文件 1: icon/AI.svg
---
```

### 字段说明

| 字段 | 说明 |
|------|------|
| `sorting-spec` | 当前文件夹的直接子项排序列表 |
| `custom-icons` | 子项的自定义图标映射 |

### 配置查找规则

1. 每个文件夹的排序配置存储在该文件夹的 `sortspec.md` 中
2. `sortspec.md` 只记录当前目录下的直接子项
3. 不递归包含子文件夹的内容
4. 子文件夹的排序由各自的 `sortspec.md` 管理

## 文件夹笔记功能

当文件夹下存在同名 `.md` 文件时：

| 操作 | 效果 |
|------|------|
| 点击文件夹 | 打开同名 `.md` 文件 + 展开目录树 |
| 目录树显示 | 同名 `.md` 文件被隐藏 |

示例：
```
项目/
├── 项目.md      ← 文件夹笔记（目录树中隐藏）
├── 笔记1.md
└── 笔记2.md
```

点击 "项目" 文件夹会：
1. 在右侧编辑器中打开 `项目.md`
2. 在目录树中展开 "项目" 的子项

## 图标系统

### emoji 图标

支持所有标准 emoji，包括复合 emoji（如 🗂️、😊 等）。

输入任意 emoji 点击"使用"即可。

### 自定义图片图标

1. 切换到 "自定义图标" Tab
2. 点击 "上传图片作为图标"
3. 选择图片并输入图标名称
4. 点击 "上传" 或 "上传并使用"

图片会保存到 vault 的 `icon/` 目录下。

**支持的文件格式：** SVG, PNG, JPG, JPEG, WebP, GIF

## 技术实现

### 核心模块

| 文件 | 功能 |
|------|------|
| `main.ts` | 插件入口，注册视图和命令 |
| `MySortView.ts` | 视图容器 |
| `DragDropTree.ts` | 拖拽树实现，核心交互逻辑 |
| `IconPickerModal.ts` | 图标选择器弹窗 |
| `SortSpecManager.ts` | sortspec.md 读写管理 |
| `RecentIconsManager.ts` | 最近使用图标管理 |
| `styles.css` | 样式定义 |

### 数据流

```
用户操作 (拖拽/图标)
    ↓
DragDropTree 处理
    ↓
SortSpecManager 保存
    ↓
sortspec.md 更新
    ↓
视图刷新
```

## 常见问题

### Q: 图标显示为 "?" 或乱码？

A: 可能是 emoji 的 surrogate pair 编码问题。请确保使用正确的 emoji 输入方式。

### Q: 拖拽后文件列表没有更新？

A: 视图会自动刷新。如果未刷新，可以手动点击"刷新"按钮。

### Q: 子文件夹的排序没有生效？

A: 确保子文件夹中也有 `sortspec.md` 文件。

### Q: 如何重置某个文件夹的排序？

A: 删除该文件夹的 `sortspec.md` 文件即可恢复默认排序。
